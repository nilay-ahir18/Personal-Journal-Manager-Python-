# Personal-Journal-Manager-Python-
The Personal Journal Manager is a simple Python-based application that allows users to maintain a digital journal. Users can add, view, search, and delete journal entries using a menu-driven interface.  This project is built using core Python concepts only, without using any external libraries or modules.
🎯 Objectives

To implement file handling in Python

To use Object-Oriented Programming (OOP) concepts

To apply exception handling for error management

To create a user-friendly menu-driven application

⚙️ Features

✍️ Add new journal entries

📖 View all saved entries

🔍 Search entries using keywords

🗑️ Delete all entries with confirmation

⚠️ Handles errors like invalid input or missing file

🧠 Concepts Used

File Handling

Read (r)

Write (w)

Append (a)

Object-Oriented Programming (OOP)

Class and methods

Exception Handling

try-except blocks

User Input Handling
